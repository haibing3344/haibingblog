1、安装lrzsz

```shell
yum install -y lrzsz
```
2、sz命令发送文件到本地

```shell
sz filename
```
3、rz命令本地上传文件到服务器：

```shell
 rz
 ```

执行该命令后，在弹出框中选择要上传的文件即可。
